# LC Dissectors

This repository holds Wireshark dissectors for Les Companions.

## Zenoh dissectors

### Installation

1. Copy or link `zenoh_scout.lua` and `zenoh_utils.lua` in ` ~/.local/lib/wireshark/plugins/`
2. Build
3. Copy or link `libzenoh.so` in ` ~/.local/lib/wireshark/plugins/4.4/epan/`
